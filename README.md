# Password Generator

Hi,

I am Feyisetan. Welcome to *STORY EMPIRE SOLUTIONS*!

The password generator project is developed to generate passwords randomly by combining letters, numbers and special characters together.

Every time the programmes is executed, a password containing letters, numbers and special characters is randomly generated, which is based on the length of the password specified by the user.
